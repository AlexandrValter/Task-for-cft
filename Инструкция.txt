Код написан в Intellij IDEA на Java SE 11
Для чтения и записи файлов используется кодировка UTF-8
Исполняемые файлы находятся по адресу: \java-testtask-cft\out\artifacts\java_testtask_cft_jar 

Можно запускать с помощью java-testtask-cft.exe или java-testtask-cft.jar, пример запуска из командной строки:
"java-testtask-cft.exe -a -s out.txt test1.txt test2.txt", где:
1. java-testtask-cft.exe или java-testtask-cft.jar: файл для запуска приложения;
2. -a или -d: вариант сортировки. Если не указать, по умолчанию будет использована сортировка по возрастанию;
3. -s или -i: тип входных данных. Для корректного выполнения программы нужно указывать обязательно;
4. out.txt - файл с результатом слияния, указывать обязательно;
5. test1.txt test2.txt - файлы с данными для обработки, указывать обязательно. Файлы с данными должны лежать в папке с исполняемым файлом.

Предполагается что файлы с данными для обработки предварительно остортированы.
Количество входных файлов может быть любым. Все они считываются параллельно по одной строке - с целью исключения переполнения оперативной памяти.
Запись в файл также происходит параллельно считыванию. Входные данные при считывании проходят проверку на указанные в ТЗ возможные ошибки:
- соответствие типу данных;
- наличие пробелов в строке;
- соответствие указанной сортировке.
Регистр учитывается при сортировке.
